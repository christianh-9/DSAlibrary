# DSAlibrary

##  
A C++ project implementing core Data Structures and Algorithms.
Each structure includes a menu system for user interaction.

- Linked List, stacks, queues
- Graphs
- Hash Tables
- Binary Search Trees

# How to run

2. Open in IDE of choice
3. Run main.cpp
